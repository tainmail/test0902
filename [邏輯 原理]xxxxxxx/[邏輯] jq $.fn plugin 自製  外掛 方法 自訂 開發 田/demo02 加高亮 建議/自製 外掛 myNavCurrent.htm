$.fn 自製 外掛

本例： 按了加current

這是按鈕 這是按鈕 這是按鈕 這是按鈕 這是按鈕 
//自行開發一個外掛  按了加current
$.fn.myNavCurrent = function(currentClass){
    return $(this).siblings().removeClass(currentClass).end().addClass(currentClass);
}



//調用外掛
$(function(){
	$(".nav a").on("click", function(){
		$(this).myNavCurrent("current")
	});
});
