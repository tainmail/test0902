參數整理
首先GreenSock一開始真的是針對Flash的Tween（補間）寫了許多好用的Class供設計師們使用。
GreenSock的v12（第十二版）中，加入了JavaScript的Class，也就是說我們可以在HTML網頁中使用TweenLite或TweenMax...等來製作動畫
功能：
css屬性動畫
css動畫時間軸
支援css3 transform 屬性
css3 3D 好像不支援(待測)
色彩屬性要用rgba 才有漸變效果
目前網路上資料有限，不過基本用法足夠。

網頁要使用這個Class，需要先import，不過HTML的指令如下： 1
<script src="src/minified/TweenMax.min.js"></script>
屬性動畫 
TweenMax.to( 目標容器,  動畫秒,  { css:{屬性:值},   delay:延遲秒,  ease:緩衝, onComplete:事件觸發}   );

TweenMax.to( $(".myBox"),   1,   { css:{width:300},   delay:1,   ease:Expo.easeOut, onComplete:fnA}  );

//注意css屬性，有兩層「{...}」
動畫時間軸 
//新增一組動畫陣列，不用設delay。播完一段，換下一段。

var oMyTimeLine = new TimelineMax

oMyTimeLine.to($(this), 1, {css:{width:300}, ease:Expo.easeOut});
oMyTimeLine.to($(this), 1, {css:{height:200}, ease:Expo.easeOut});
oMyTimeLine.to($(this), 1, {css:{opacity:0.3}, ease:Expo.easeOut});
oMyTimeLine.to($(this), 1, {css:{width:100, height:50, opacity:1}, ease:Expo.easeOut});
 

緩衝
ease: Bounce.easeOut 彈性
ease: Circ.easeOut 等速(看不出來)
ease: Elastic.easeOut 電到
ease: Expo.easeOut 快>慢
ease: Sine.easeOut 等速(看不出來)
  easeIn 開頭作用
easeOut 結尾作用
easeInOut 頭尾作用
 

callback 事件觸發
onStart : fn
 動畫開始
onUpdate : fn 動畫過程
onComplete : fn 動畫結束
